# Nöbet Çizelgesi Uygulaması

Basit ve pratik nöbet planlama uygulaması.

## Özellikler
- Adil nöbet dağıtımı
- Kıdeme göre sıralama
- Uzmanlar tek rütbe kabul edilir
- Subay / astsubay sistemi desteklenir
